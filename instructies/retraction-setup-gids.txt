WAT IS RETRACTION?
------------------
Retraction trekt filament terug bij travel moves om stringing te voorkomen.
Direct Drive: 0.2-1.0mm
Bowden: 2-10mm
TPU: 0-1.2mm (lage snelheid!)


CURA INSTELLEN
================================================================================

STAP 1: Visibility inschakelen
-------------------------------
1. Klik rechtsboven op "Custom" dropdown
2. Klik op "All" of zoek "retraction"
3. Vink aan:
   ☑ Retraction Distance
   ☑ Retraction Speed
   ☑ Retraction Retract Speed (optioneel)
   ☑ Retraction Prime Speed (optioneel)

STAP 2: Retraction Distance instellen
--------------------------------------
Zoek "Retraction Distance" in settings

Direct Drive starten met:    0.6mm
Bowden starten met:          6.0mm
TPU starten met:             0.3mm

STAP 3: Retraction Speed instellen
-----------------------------------
Zoek "Retraction Speed" in settings

Direct Drive:                45 mm/s
Bowden:                      25 mm/s
TPU:                         25 mm/s

STAP 4: Optionele settings
---------------------------
"Retraction Retract Speed" = snelheid bij intrekken
"Retraction Prime Speed" = snelheid bij terugduwen

Meestal beide gelijk houden aan Retraction Speed!


ORCASLICER INSTELLEN
================================================================================

STAP 1: Filament Settings openen
---------------------------------
1. Klik bovenin op "Filament Settings" tab (icoon met filament rol)
2. Of: Printer Settings > Filament

STAP 2: Retraction instellingen vinden
---------------------------------------
Scroll naar sectie: "Retraction"

Je ziet:
- Length (retraction distance)
- Lift Z (optioneel)
- Speed
  - Retraction Speed
  - Deretraction Speed

STAP 3: Length (afstand) instellen
-----------------------------------
Direct Drive:                0.6mm
Bowden:                      6.0mm
TPU:                         0.3mm

STAP 4: Speed instellen
------------------------
Retraction Speed:
  Direct Drive:              45 mm/s
  Bowden:                    25 mm/s
  TPU:                       25 mm/s

Deretraction Speed:
  Meestal GELIJK aan Retraction Speed!

STAP 5: Settings opslaan
-------------------------
1. Klik rechtsboven op diskette icoon
2. Of: File > Save


RETRACTION TOWER TESTEN
================================================================================

Na het instellen van deze basis waarden:

1. Download retraction tower STL van werkplaatsmarc.be/retraction.html
2. Slice met bovenstaande settings
3. Print de tower
4. Bekijk tussen de twee kolommen welke sectie minste stringing heeft
5. Pas Retraction Distance aan op basis van beste sectie


TROUBLESHOOTING
================================================================================

TE VEEL STRINGING?
------------------
→ Verhoog Retraction Distance met 0.2-0.5mm stappen
→ Verlaag nozzle temperatuur met 5°C
→ Verhoog Travel Speed
→ Check of filament droog is

UNDER-EXTRUSIE NA TRAVEL?
--------------------------
→ Verlaag Retraction Distance met 0.2mm stappen
→ Verhoog Retraction Speed
→ Check extruder spanning (te strak = slippen)
→ Check bowden tube (slijtage?)

BLOBS BIJ START NIEUWE LAAG?
-----------------------------
→ Retraction Distance iets verhogen
→ "Retraction at Layer Change" activeren (Cura)
→ Z-Hop activeren (optioneel)


BELANGRIJKE OPMERKINGEN
================================================================================

✓ Start ALTIJD met deze basis waardes
✓ Wijzig in kleine stappen (0.2-0.5mm)
✓ Test elke wijziging met een print
✓ Noteer je beste settings per filament type
✓ Verschillende filament merken = verschillende settings!

✗ NOOIT boven 2mm voor Direct Drive
✗ NOOIT boven 15mm voor Bowden
✗ TPU heel voorzichtig met retraction (vaak 0mm beste!)
