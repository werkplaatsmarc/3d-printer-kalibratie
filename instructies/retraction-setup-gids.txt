METHODE 1: CALIBRATION SHAPES PLUGIN (MAKKELIJKST!)
================================================================================

INSTALLATIE
-----------
1. Open Cura
2. Klik: Marketplace (bovenin menu)
3. Zoek: "Calibration Shapes"
4. Klik: Install (van 5@xes)
5. Herstart Cura

RETRACTION TOWER GENEREREN
---------------------------
1. Klik: Extensions > Calibration Shapes > Add a Retraction Tower
2. Plugin genereert automatisch de tower!
3. Slice normaal
4. Print en evalueer

WAAROM DIT DE BESTE METHODE IS:
- Automatisch tower model
- Geen handmatig werk
- Direct klaar voor printen


METHODE 2: POST PROCESSING SCRIPT (INGEBOUWD IN CURA)
================================================================================

DEZE METHODE ALS JE CALIBRATION SHAPES NIET WILT INSTALLEREN

STAP 1: Download Tower STL
---------------------------
Download van werkplaatsmarc.be/retraction.html:
- retraction-tower-direct.stl (Direct Drive)
- retraction-tower-bowden.stl (Bowden)
- retraction-tower-tpu.stl (TPU)

STAP 2: Import en Slice
------------------------
1. Sleep STL naar Cura
2. Zet retraction settings:
   - Enable Retraction: AAN
   - Retraction Distance: Start waarde (bijv. 2mm voor bowden)
   - Retraction Speed: 25mm/s (bowden) of 45mm/s (direct drive)
3. Slice de tower

STAP 3: Post Processing Script Toevoegen
-----------------------------------------
1. Ga naar: Extensions > Post Processing > Modify G-Code
2. Klik: Add a script
3. Kies: RetractTower
4. Stel in:

   VOOR DIRECT DRIVE:
   VOOR DIRECT DRIVE:
   - Command: Distance
   - Starting Value: 0.2
   - Value Increment: 0.2
   - Change Layer: elke 25 layers (voor 0.2mm layer height = 5mm)
   
   VOOR BOWDEN:
   - Command: Distance  
   - Starting Value: 2
   - Value Increment: 2
   - Change Layer: elke 25 layers (voor 0.2mm layer height = 5mm)

5. Klik: Close
6. Slice opnieuw (Cura past nu G-code automatisch aan!)
7. Print!

BELANGRIJKE OPMERKING:
De script werkt ALLEEN als je "Enable Retraction" hebt aangezet in Cura
en een retraction distance > 0mm hebt ingesteld!


TOWER EVALUEREN
================================================================================

LEES VAN ONDER NAAR BOVEN
--------------------------
1. Onderste sectie = Laagste retraction waarde
2. Check tussen de twee kolommen voor stringing draden  
3. Welke sectie heeft MINSTE stringing?
4. Noteer die retraction waarde

BESTE WAARDE TOEPASSEN IN CURA
-------------------------------
1. Zoek "Retraction Distance" in settings
2. Vul de beste waarde in
3. Save als nieuw filament profiel


TROUBLESHOOTING
================================================================================

TE VEEL STRINGING?
------------------
→ Verhoog Retraction Distance met 0.2-0.5mm stappen
→ Verlaag nozzle temperatuur met 5°C
→ Verhoog Travel Speed
→ Check of filament droog is

UNDER-EXTRUSIE NA TRAVEL?
--------------------------
→ Verlaag Retraction Distance met 0.2mm stappen
→ Verhoog Retraction Speed  
→ Check extruder spanning (te strak = slippen)
→ Check bowden tube (slijtage?)

BLOBS BIJ START NIEUWE LAAG?
-----------------------------
→ Retraction Distance iets verhogen
→ "Retraction at Layer Change" activeren
→ Z-Hop activeren (optioneel)
