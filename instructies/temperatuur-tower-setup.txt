WAT IS EEN TEMPERATUUR TOWER?
------------------------------
Test verschillende temperaturen in één print om de optimale temperatuur te 
vinden voor elk specifiek filament. Elke sectie print op een andere temperatuur.


METHODE 1: CALIBRATION SHAPES PLUGIN (MAKKELIJKST!)
================================================================================

INSTALLATIE
-----------
1. Open Cura
2. Klik: Marketplace (rechtsboven)
3. Zoek: "Calibration Shapes"
4. Klik: Install (van 5@xes)
5. Herstart Cura

TEMPERATUUR TOWER GENEREREN
----------------------------
1. Klik: Extensions > Part for calibration
2. Kies je filament type:
   - Add a PLA Temp Tower
   - Add a PETG Temp Tower
   - Add a ABS Temp Tower
   (of gebruik "Add a Temp Tower" voor custom range)

3. De tower wordt automatisch toegevoegd!

4. Ga naar: Extensions > Post Processing > Modify G-Code
5. Je ziet automatisch "TempFanTower" script toegevoegd
6. Controleer de instellingen (normaal al correct!)

7. Slice en print!

WAAROM DIT DE BESTE METHODE IS:
- Automatisch tower model
- Automatisch post processing script
- Geen handmatig rekenwerk
- Direct klaar voor printen


METHODE 2: POST PROCESSING SCRIPT (HANDMATIGE STL)
================================================================================

DEZE METHODE ALS JE CALIBRATION SHAPES NIET WILT INSTALLEREN

STAP 1: Download Tower STL
---------------------------
Download van werkplaatsmarc.be/temperatuur.html:
- temp-tower-PLA.stl (180-220°C)
- temp-tower-PETG.stl (220-250°C)
- temp-tower-ABS.stl (210-250°C)
- temp-tower-TPU.stl (210-230°C)

STAP 2: Import in Cura
-----------------------
1. Sleep STL naar Cura
2. Slice NIET meteen!

STAP 3: Post Processing Script Toevoegen
-----------------------------------------
1. Ga naar: Extensions > Post Processing > Modify G-Code
2. Klik: Add a script
3. Kies: ChangeAtZ (meerdere keren toevoegen!)

TOWER EVALUEREN
================================================================================

CHECK PER SECTIE:
-----------------
✓ Overhang kwaliteit (schuine wanden)
✓ Bridging prestatie (hangende delen)
✓ Layer adhesie (laag hechting)
✓ Oppervlakte kwaliteit (gladheid)
✓ Stringing (draadjes tussen delen)

BESTE SECTIE = OPTIMALE TEMPERATUUR!


BELANGRIJKE TIPS
================================================================================

✓ Calibration Shapes plugin is MAKKELIJKSTE methode
✓ Vergeet niet scripts te verwijderen voor volgende print!
✓ Test elke nieuwe filament rol (variatie tussen batches)
✓ Noteer beste temperatuur per merk/type

✗ Vergeet niet om na de test de scripts te verwijderen!
✗ Check altijd de G-code preview voor je print
